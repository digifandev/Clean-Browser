# ⚡ Brave Ultra Low-End Optimization

Ez a registry tweak csomag a Brave böngésző optimalizálására készült nagyon gyenge gépekhez.

## 💻 Tesztelt konfiguráció
- CPU: Intel N100
- RAM: 4 GB
- OS: Windows 10/11 (64-bit)

## 🚀 Mit csinál?

- Kikapcsolja az összes felesleges Brave funkciót
- Letiltja a telemetriát
- Csökkenti a RAM és CPU használatot
- Optimalizálja a böngészőt low-end gépekre

## 📉 Eredmény

- Kevesebb memóriahasználat (~150-300MB)
- Gyorsabb böngészés
- Stabilabb működés gyenge hardveren

## ⚙️ Telepítés

1. Másold ki a `.reg` fájl tartalmát
2. Mentsd el pl. `brave-optimization.reg` néven
3. Kattints rá duplán
4. Indítsd újra a gépet vagy a Brave-et

## ⚠️ Figyelmeztetés

Ez a tweak:
- Kikapcsol bizonyos funkciókat (pl. jelszó mentés, Brave Rewards)
- Haladó felhasználóknak ajánlott

## 🧠 Ajánlott extra

- Használj kevesebb extensiont
- Kapcsold ki a felesleges tabokat
- Használj uBlock Origin-et (light módban)

## 📌 Cél

Minél jobb teljesítmény elérése ultra gyenge gépeken.
